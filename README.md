# ai-agent

CLI simples em Python que chama a API do Google Gemini para gerar uma resposta de texto.

## Requisitos

- Python 3.12+
- [uv](https://github.com/astral-sh/uv)

## Configuração

Crie um arquivo `.env` na raiz do projeto:

```sh
GEMINI_API_KEY='sua_chave_aqui'
```

## Instalação

```sh
uv sync
```

## Execução

```sh
uv run main.py
```
